# 博客变现助手 (Blog Monetization Helper)

一站式博客变现工具平台，助力内容创作者实现 Google AdSense 变现。

## 功能

1. 文本处理工具
   - 字数统计（字符/中文字数/英文单词数）
   - 标题生成器
   - 关键词提取器

2. SEO 优化工具
   - 标题优化（50-60 字符）
   - 描述优化（130-160 字符）
   - 关键词提取
   - Meta 标签生成

3. 格式化工具
   - Markdown 预览
   - HTML 预览

4. AdSense 配置
   - 广告代码输入
   - 代码预览
   - 本地保存（localStorage）

5. 博客资源
   - AdSense 申请指南
   - 优化技巧
   - SEO 策略
   - 变现策略

## 技术栈

- HTML5（语义化）
- CSS3（响应式）
- JavaScript（ES6+, 本地存储）

## 部署

### 方式 1：本地测试
```bash
cd /home/admin/projects/opencode/blog-monetize-helper
open index.html
```

### 方式 2：GitHub Pages
1. 创建仓库：https://github.com/new
2. 仓库名：blog-monetize-helper
3. 上传 index.html
4. 设置 GitHub Pages

### 方式 3：Vercel 部署
```bash
npm i -g vercel
vercel login
vercel --prod
```
